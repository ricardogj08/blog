🗃 git-annex

git-annex permite administrar archivos grandes con git sin almacenar el contenido del archivo en el repositorio, con la posibilidad de sincronizar, respaldar y archivar tus datos, con o sin conexión. Los datos se mantienen seguros y protegidos gracias al sistema de verificación y cifrado.

git-annex está diseñado para usuarios de git que aman la línea de comandos. Para los demás usuarios, git-annex cuenta con un asistente basado en web para sincronizar carpetas fácilmente.
